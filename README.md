# 🎓 Attendance Management System

A **console-based Attendance Management System** developed using **Java** and **Object-Oriented Programming (OOP)** principles. This project allows users to manage student attendance through a simple console interface.

---

## ✨ Features

- 🔐 User Signup
- 🔑 User Login
- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by ID
- ✅ Mark Student Attendance
- 📊 View Attendance Status
- 🚪 Logout

---

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/krishdhiman9/Attendance_Management_System.git
```

2. Open the project in **IntelliJ IDEA**.

3. Run src/com/attendance/Main.java.

4. Follow the on-screen menu.
